Development:Creating a shopping platform in React

Overview

This project is a list-based interactive interface built in React. It comprises interface componenents and ties these components to an internal data state. The interface allows users to select items out of a list, and then aggregate them.

It also supports filtering and sorting options.

Architecture

The list-based interface handles any combination of filters. All the filters and sorting methods work together so that the user is able to narrow down to a particular type of product. Users can revert back to the original state without refreshing the page. The aggregator allows users to add and remove stuff from the cart.
